1D-CNN Architecture

Model 1D Convolutional Neural Network (1D-CNN) digunakan untuk mempelajari pola temporal pada sinyal EKG. Arsitektur ini mampu mengenali bentuk gelombang abnormal yang berkaitan dengan aritmia.

  • ✅ Feature Extraction
  • ✅ Convolution Layer
  • ✅ Max Pooling Layer
  • ✅ Fully Connected Layer
  • ✅ Softmax Classification

MIT-BIH Dataset

Model AI dilatih menggunakan MIT-BIH Arrhythmia Database yang merupakan salah satu dataset standar internasional untuk penelitian deteksi aritmia.
