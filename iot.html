IoT Pipeline Architecture

Sistem Smart EKG menggunakan arsitektur Internet of Things (IoT) untuk melakukan monitoring kesehatan jantung secara realtime dari perangkat sensor menuju cloud database dan aplikasi monitoring.

Firebase Cloud

Database cloud digunakan untuk menyimpan data monitoring pasien dan hasil rekaman EKG secara realtime.

🔒

HTTPS Security

Komunikasi data menggunakan protokol HTTP/HTTPS untuk menjaga keamanan data monitoring pasien.

📱

Mobile Monitoring

Aplikasi monitoring digunakan untuk menampilkan BPM, grafik EKG, dan hasil deteksi aritmia.

Fitur IoT

  • ✅ Realtime ECG Monitoring
  • ✅ ESP32 Wireless Gateway
  • ✅ Cloud Synchronization
  • ✅ Firebase Realtime Database
  • ✅ Distributed System Support
  • ✅ Offline Buffer Storage

Distributed System

Sistem dirancang agar tetap dapat berjalan dalam kondisi offline maupun online. Ketika koneksi internet terputus, data akan disimpan sementara secara lokal dan dikirim kembali ke cloud ketika koneksi tersedia.

Offline Mode

Data disimpan sementara pada local buffer storage.

Online Mode

Data otomatis disinkronkan ke Firebase Cloud.
